Download the Chinese small Vosk model zip and place it in this folder if you
want to bundle it inside the app.

Default (Chinese, small):
- vosk-model-small-cn-0.22.zip

If the file is missing, the app will try to auto-download it on first run.

Download from:
https://alphacephei.com/vosk/models

Make sure the filename matches the path in `lib/services/speech_to_text.dart`.
